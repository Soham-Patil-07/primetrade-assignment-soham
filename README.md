# Trader Performance Analysis Using Bitcoin Fear & Greed Index

## Objective
Analyze the relationship between Bitcoin market sentiment and trader performance using Hyperliquid historical trading data and the Fear & Greed Index.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Analysis Performed
- Average PnL by Sentiment
- Win Rate Analysis
- Coin-wise Profitability
- Trade Distribution Analysis

## Key Findings
- Extreme Greed showed the highest average profit.
- Traders achieved higher win rates during Extreme Greed periods.
- Market sentiment influences trader profitability and success rates.
